---
title: Tutorial - First Successful Run
description: End-to-end tutorial from onboarding to validated run output.
---

1. Complete [Quickstart](/quickstart).
2. Confirm `GET /health/ready` returns `ready=true`.
3. Create a run through `POST /api/v1/runs`.
4. Validate tasks and graph endpoints.
5. Capture event evidence from `/api/v1/events`.

This tutorial is optimized for first confidence, not full production hardening.
