## Gemeinsame Pre-Flight-Checks

1. Git ist installiert und im PATH.
2. Go >= 1.24 ist verfügbar.
3. Node >= 20 ist verfügbar.
4. Port `59013` kann lokal gebunden werden.

## Verifikation

```bash
curl -fsS http://127.0.0.1:59013/health/ready
```

Erwartet: `"ready":true`.
