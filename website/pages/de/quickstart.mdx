<h2 id="schritt-1">Schritt 1: Klonen und Onboarding</h2>

<CommandTabs
  macos={`git clone <repo-url> BIOMETRICS\ncd BIOMETRICS\n./biometrics-onboard`}
  linux={`git clone <repo-url> BIOMETRICS\ncd BIOMETRICS\n./biometrics-onboard`}
  windowsWSL={`git clone <repo-url> BIOMETRICS\ncd BIOMETRICS\n./biometrics-onboard`}
/>

<VerifyBlock title="Verifiziere Onboarding-Report" expected='"ready":true oder nur warnings'>
  Prüfe <code>.biometrics/onboard/report.json</code> und behebe alle Blocker vor dem nächsten Schritt.
</VerifyBlock>

<h2 id="schritt-2">Schritt 2: Control Plane starten</h2>

<CommandTabs
  macos={`cd biometrics-cli\ngo run ./cmd/controlplane`}
  linux={`cd biometrics-cli\ngo run ./cmd/controlplane`}
  windowsWSL={`cd biometrics-cli\ngo run ./cmd/controlplane`}
/>

<VerifyBlock title="Verifiziere Health" expected='{"status":"ok"}'>
  In einem zweiten Terminal: <code>curl -fsS http://127.0.0.1:59013/health</code>
</VerifyBlock>

<h2 id="schritt-3">Schritt 3: Ersten Run erstellen</h2>

<CommandTabs
  macos={`curl -fsS -X POST http://127.0.0.1:59013/api/v1/runs \\\n  -H "Content-Type: application/json" \\\n  -d '{"project_id":"biometrics","goal":"quickstart run","mode":"autonomous"}'`}
  linux={`curl -fsS -X POST http://127.0.0.1:59013/api/v1/runs \\\n  -H "Content-Type: application/json" \\\n  -d '{"project_id":"biometrics","goal":"quickstart run","mode":"autonomous"}'`}
  windowsWSL={`curl -fsS -X POST http://127.0.0.1:59013/api/v1/runs \\\n  -H "Content-Type: application/json" \\\n  -d '{"project_id":"biometrics","goal":"quickstart run","mode":"autonomous"}'`}
/>

<VerifyBlock title="Verifiziere Run-ID" expected='"id":"run-..."'>
  Die Antwort muss eine nicht-leere Run-ID und einen initialen Status enthalten.
</VerifyBlock>

<div className="ops-note">
